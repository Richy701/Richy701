#### What I'm shipping

**[Free Trade Journal](https://www.freetradejournal.com/)** — Trade journaling platform. P&L tracking, equity curves, broker CSV imports, dark/light themes, Stripe-powered Pro tier with quarterly recaps. Live with paying users.

**Lapsly** — iOS subscription & expiry tracker. Custom design system, NavigationStack architecture, onboarding flow. *(SwiftUI + SwiftData)*

**Depuis** — iOS "anti-habit" tracker for irregular tasks — things you do, but not on a schedule. *(SwiftUI + SwiftData)*

**Not Now** — ADHD task manager built around low-friction capture and flexible prioritisation. *(SwiftUI + SwiftData)*

**[Regal Est.](https://regalest.com/)** — Streetwear brand site. 3D spinning logo, grunge textures, colour-cycling animations.
